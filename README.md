# WTR- Wifi to Radio 模块

---

## 概述：

WTR是由CUAV全新推出的一款应用于 无人机地面站端 WIFI转数传的转发模块，将地面端数传数据转换为无线WIFI数据，实现电脑或者手机 WIFI无线连接无人机，代替传统的蓝牙转数传模块，实现更高可靠性和便捷性

## 无线转发互联

在特殊无人机作业时，往往需要远距离的通讯链路数传，而传统的远距离数传只能通过有线USB进行连接，这样会使操作人员在临时架设地面通讯站时很受约束。使用WTR后，这些问题一却不复返，WTR将地面端的数传TTL信号，转换成WIFI 信号，通过手机或者电脑进行连接，使地面数传直接可以架设在高处，从而无人机通讯距离更远更稳定
